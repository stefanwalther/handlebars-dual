# handlebars-dual

> Handlebars helpers to work for both, markdown and html output.

Install with [npm](https://www.npmjs.com/)

```bash
npm i handlebars-dual --save
```

## Usage

```js
var handlebarsDual = require('handlebars-dual');
```
